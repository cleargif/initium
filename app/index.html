Initium (Project Boilerplate)

Simple front-end boilerplate and workflows.

Boilerplater repository to kick start your web project without wasting your valuable time. Initium contains all useful task from compiling, testing and deploying.

Available tools & tasks

  • Mobile First
  • Grunt.js
    • Less
    • Require.js
    • CoffeeScript
    • Mocha (Unit Test)
    • Deployment / FTP to server
    • Livereload
    • usemin
    • JSLint
  • Twitter Bootstrap 3
  • HTML5 Boilerplate
  • Require.js
  • Less / CSS
  • CSS3 Mixins
  • Bower (Package Manager)
  • Wraith (screenshot comparison tool)
